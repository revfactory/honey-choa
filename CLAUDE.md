## 하네스: 꿀초아tv 댄스 교육 사이트

**목표:** YouTube Data API v3로 꿀초아tv(@youzin, `UCNExEo8zrCEXs6DA6yeEcoA`) 채널의 영상·숏츠를 수집·큐레이션하여, 모던하고 인터랙티브한 댄스 교육 사이트를 구축·운영한다.
