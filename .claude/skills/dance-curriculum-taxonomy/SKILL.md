# Dance Curriculum Taxonomy — 댄스 교육 분류 체계

## 분류 철학

분류는 운영자가 아닌 **학습자의 질문**에 답하기 위해 존재한다:
"내 수준에 맞는 게 뭐지?" (난이도) / "이 노래 안무 있나?" (장르·곡) / "뭐부터 보지?" (트랙)

채널 실데이터를 보기 전에 분류를 확정하지 마라. 아래 체계는 출발점이며, 전체 제목 목록을 스캔해 채널의 실제 시리즈·포맷에 맞게 보정한 후 적용한다.

## 분류 축

### 1. 콘텐츠 유형 (type) — 필수, 단일값

| 값 | 판별 단서 |
|----|----------|
| `tutorial` | "배우기", "강좌", "튜토리얼", "거울모드", "느리게", 구간 설명 |
| `choreography` | "안무", "커버", "cover", 풀버전 댄스 |
| `challenge` | "챌린지", "challenge" (주로 숏츠) |
| `basics` | "기초", "스트레칭", "웜업", "동작 연습" |
| `vlog_etc` | 일상, 비하인드, 공지 등 비학습 콘텐츠 |

### 2. 장르 (genre) — 필수, 단일값

`kpop` / `jpop` / `pop` / `hiphop` / `jazz` / `etc` — 곡명·아티스트명이 제목에 있으면 그것이 1차 단서. 모호하면 `etc`.

### 3. 난이도 (difficulty) — tutorial/basics/choreography에만 부여

| 값 | 기준 |
|----|------|
| `beginner` | 기초 동작, 느린 템포, "왕초보"·"기초" 명시 |
| `intermediate` | 일반 안무 커버 기본값 — 명시 단서 없으면 이 값 |
| `advanced` | 고속·고난도 명시, 긴 풀안무, "고급" 단서 |

판별 불가가 기본값(intermediate)과 다름에 주의 — 단서가 전혀 없는 비학습 콘텐츠는 난이도를 부여하지 않는다.

### 4. 곡 정보 (song) — 선택

`{ title, artist }` — 제목에서 추출. 「」, '', 대괄호 패턴을 우선 파싱. 불확실하면 생략(추측 금지).

## 분류 규칙

1. **모든 분류에 근거를 남긴다**: `evidence: "제목에 '기초 스트레칭'"` — 근거 없는 분류는 검수와 수정이 불가능하다.
2. **추측 금지**: 단서가 불충분하면 `type: "unclassified"`로 두고 별도 목록으로 보고한다. 오분류된 난이도는 학습자가 좌절하거나 지루해하는 직접 원인이 된다.
3. **분포 점검**: 분류 완료 후 축별 분포를 산출한다. 한 값에 80% 이상 쏠리면 그 축은 변별력이 없다 — 세분화하거나 축을 교체한다.

## 영상 ↔ 숏츠 매칭

같은 안무의 풀버전(영상)과 챌린지(숏츠)를 연결하면 "숏츠로 발견 → 풀버전으로 학습" 동선이 생긴다.

- 매칭 단서: 곡명·아티스트 일치 > 제목 핵심 토큰 일치 > 게시일 근접(±14일)
- 결과 필드: `relatedIds: string[]` (양방향으로 기록)
- 불확실한 매칭은 연결하지 않는다 — 잘못된 연결은 미연결보다 나쁘다

## 커리큘럼 트랙 구성

트랙 = 순서가 있는 학습 경로. 최소 3개, 콘텐츠가 5건 미만인 트랙은 만들지 않는다.

```json
{
  "trackId": "kpop-starter",
  "title": "K-POP 입문 트랙",
  "description": "기초 동작부터 첫 안무 완성까지",
  "steps": [{ "videoId": "...", "note": "선택 사유" }]
}
```

구성 원칙: ① 첫 스텝은 항상 beginner ② 난이도 단조 증가 ③ 같은 장르 내에서 구성 ④ 스텝 사이 난이도 점프 금지 (beginner 다음에 advanced 배치 불가)

## 산출 형식

`data/curated.json` — 원본을 복제하지 않고 videoId 참조 + 분류 필드만 추가:

```json
{
  "generatedAt": "ISO8601",
  "items": [{ "videoId": "...", "type": "tutorial", "genre": "kpop",
              "difficulty": "beginner", "song": {...}, "relatedIds": [],
              "evidence": "...", "manual": false }],
  "tracks": [...],
  "stats": { "byType": {}, "byGenre": {}, "byDifficulty": {}, "unclassified": 0 }
}
```

`manual: true`는 사람이 보정한 항목 — 자동 재분류 시 절대 덮어쓰지 않는다.
